# Scripts

Utility scripts used to maintain this repository.

## sync_all_scripts.sh

Re-downloads and replaces all upstream Vast.ai script files listed in
[files/vast.ai/README.md](../files/vast.ai/README.md) into
[files/vast.ai](../files/vast.ai/).

Usage:

```bash
./scripts/sync_all_scripts.sh
```

Dry run:

```bash
./scripts/sync_all_scripts.sh --dry-run
```
